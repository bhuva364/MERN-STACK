HTTP \&HTTPS - THE LANGUAGE OF THE WEB



HTTP METHODS:-

* GET - READ DATA FROM SERVER
* POST - SEND NEW DATA TO SERVER(CREATE)
* PUT - FULL UPDATE (IN  ONE ROW)
* DELETE - REMOVE DATA FROM SERVER
* PATCH - PARTIALLY UPDATE EXISTINIG DATA



HTTP STATUS CODE:-

1. 200 - OK
2. 201 - CREATED
3. 301 - MOVED
4. 400 - BAD REQUEST
5. 401 - UNAUTHORIZED
6. 403 - FORBIDDEN
7. 404 - NOT FOUND
8. 500 - SERVER ERRRO
